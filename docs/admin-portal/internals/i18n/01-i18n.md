# i18n (다국어) 구조

Vue-i18n 기반. 한국어(ko)가 기본 언어, 영어(en) 번역 파일 존재하지만 현재 비활성화 상태. user-portal과 동일한 구조.

## 설정 — `src/lang/index.js`

```javascript
const i18n = new VueI18n({
  locale: localStorage.getItem('language') || 'ko',
  messages,
  silentTranslationWarn: true  // 누락된 번역 키 경고 억제
})
```

- 언어 설정은 localStorage `language` 키에 저장, 세션 간 유지.
- Element UI 한국어 로컬라이제이션(`element-ui/lib/locale/lang/ko`) 적용.
- 영어 번역 파일은 import 주석 처리, 미사용.

## 번역 파일 구조

`src/lang/menus/` 아래 `ko/`와 `en/` 폴더에 동일한 구조로 파일 존재:

| 파일 | 용도 |
|------|------|
| `page.js` | 페이지 제목/레이블 |
| `message.js` | 안내/경고 메시지 |
| `list.js` | 테이블 컬럼 레이블 |
| `button.js` | 버튼 텍스트 |
| `form.js` | 폼 레이블 |
| `validate.js` | 유효성 검사 메시지 |
| `modal.js` | 모달 다이얼로그 텍스트 |
| `disk.js` | 디스크 관리 전용 텍스트 |

## 사용 방식

```vue
<!-- 템플릿 -->
{{ $t('button.common.search') }}

<!-- 스크립트 -->
this.$t('message.common.saveComplate')
```

## 언어 전환

Vuex `app` 모듈의 `setLanguage` 액션으로 처리:
1. localStorage에 언어 코드 저장
2. `<html lang>` 속성 업데이트
3. 세션 코드(`sessionStorage.codes`) 삭제로 코드 목록 재조회 유도

현재 배포 환경에서는 한국어만 활성화.
