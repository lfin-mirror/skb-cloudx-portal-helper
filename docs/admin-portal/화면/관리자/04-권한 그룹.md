# 권한 그룹 (T1202)

TA 전용 (SA 메뉴에 없음). 관리자 계정에 할당할 권한 그룹 관리.

경로: /admin-setting/group

컴포넌트: views/adminSetting/GroupAdminSetting.vue

권한 그룹 목록 조회, 등록, 수정. 권한 초기화 기능. 각 그룹에 메뉴별 접근 권한 설정.

## 테이블 컬럼

| 컬럼 | 필드 |
|------|------|
| 권한 그룹명 | `adm_grp_nm` |
| 설명 | `descp` |
| 생성자 | `reg_conn_id` |
| 생성일 | `reg_ts` |
| 수정자 | `mod_conn_id` |
| 수정일 | `mod_ts` |

## SA 메뉴 관리와의 관계

SA의 메뉴/API/기능 관리(A1208)에서 정의한 메뉴 트리에 대해, TA의 권한 그룹에서 그룹별 접근 가능 메뉴를 설정.

```
SA: 메뉴 정의 (A120801) → 기능 매핑 (A120803) → API 매핑 (A120802)
TA: 권한 그룹 (T1202) → 메뉴별 접근 권한 설정
TA: 관리자 계정 (T1201) → 권한 그룹 할당
```
