# 관리자 계정 (A1201, T1201)

경로: /admin-setting/index

컴포넌트: views/adminSetting/AdminSetting.vue / AdminSettingDetail.vue

SA/TA 공통. 관리자 계정 CRUD.

## 연관 관계

- [권한 그룹](04-권한%20그룹.md) — TA가 관리자 계정에 권한 그룹 할당
- [메뉴-API-기능 관리](03-메뉴-API-기능.md) — SA가 RBAC 구조 정의
- [인증 정책](../정책/01-인증%20정책.md) — TA가 관리자 계정에 인증 정책 할당

## SA vs TA 차이

| 항목 | 세부 항목 | SA | TA |
|------|----------|----|----|
| **목록 API** | — | `GET /v1/user/accounts/sup` | `GET /v1/user/accounts/admin` |
| **테이블 컬럼** | 상태 (`acct_sts_cd`) | 표시 | 표시 |
| | 계정 ID (`acct_conn_id`) | 표시 | 표시 |
| | 권한 그룹 (`adm_grp_nm`) | 비노출 | 표시 |
| | 계정명 (`acct_nm`) | 표시 | 표시 |
| | 전화번호 (`mob_no`) | 표시 | 표시 |
| | 생성일 (`reg_ts`) | 표시 | 표시 |
| | 사용 기간 (`acct_vlid_dt`) | 표시 | 표시 |
| | 최근 접속 (`lst_conn_tm`) | 표시 | 표시 |
| **상세/등록 폼** | 계정 상태 (`page.admin.label.accountStatus`) | 편집 가능 | 편집 가능 |
| | ↓ **grpTypCd = 'U001TNT' (TA)일 때만 노출** | | |
| | 권한 그룹 (`page.admin.label.authorityGroup`) | 비노출 | 필수 셀렉트 |
| | 인증 정책 (`page.admin.label.certPolicy`) | 비노출 | 필수 — 모달 선택 |
| | ↓ **신규 등록 시에만 편집 가능** | | |
| | 계정 ID (`acct_conn_id`) | 편집 가능 + 중복 체크 버튼 | 동일 |
| | 비밀번호 (`passwd_data`) | 편집 가능 | 동일 |
| | ↓ **공통 필드 (SA/TA 동일)** | | |
| | 계정명 (`acct_nm`) | 편집 가능 | 편집 가능 |
| | 이메일 (`email`) | 편집 가능 ("ID와 동일" 체크박스) | 편집 가능 |
| | 전화번호 (`mob_no`) | 편집 가능 | 편집 가능 |
| | 사용 기간 (`acct_vlid_stt_dt` / `end_dt`) | 편집 가능 | 편집 가능 |
| | SMS 알림 수신 (`sms_recv_yn`) | 편집 가능 | 편집 가능 |
| | 이메일 알림 수신 (`email_recv_yn`) | 편집 가능 | 편집 가능 |
| | 개인정보 조회 권한 (`permission`) | 편집 가능 | 편집 가능 |
| | 조직명 (`ognz_nm`) | 편집 가능 | 편집 가능 |
| | 부서명 (`blng_dept_nm`) | 편집 가능 | 편집 가능 |

SA는 최상위 권한이므로 그룹/정책 없이 계정 생성. TA는 반드시 지정.

## 버튼 동작

| 버튼 | 클릭 시 동작 |
|------|------------|
| **ID 중복 체크** | `POST` 중복 확인 (신규 등록 시만 노출) |
| **인증 정책 선택** (TA) | 모달 열기 → 인증 정책 목록에서 선택 |
| **저장** (등록) | `POST /v1/user/accounts/admin` |
| **저장** (수정) | `PUT /v1/user/accounts/admin/{id}` |
| **삭제** | 확인 팝업 → `DELETE /v1/user/accounts/admin/{id}` |
