# 접근 통제 / 외부 연동 / Push 메시지 (T06)

TA 전용 (SA 메뉴에 없음). 컴포넌트 내부에 `isSuperAdmin` 분기 없음.

## 하위 메뉴 구조

```
서비스 (T06)
├── 접근 통제 (T0604)
│   ├── User 접근 통제 (T060401)
│   └── Admin 접근 통제 (T060402)
├── 외부 연동 (T0605)
│   ├── AD 연동 (T060501)
│   ├── 네트워크 스토리지 연동 (T060502)
│   ├── 이메일 연동 (T060503)
│   └── OCTATCO 연동 (T060504)
└── Push 메시지 (T0606)
    ├── 메시지 발송 (T060601)
    └── 발송 내역 (T060602)
```

## 접근 통제 (T0604)

사용자/관리자의 포털 접근을 IP·ID 기반으로 차단/해제.

### User 접근 통제 (T060401)

경로: /service-manage/access-control/histories

컴포넌트: views/service/accessControl/AccessControlHistories.vue

| 컬럼 | 필드 | 설명 |
|------|------|------|
| 타입 | `acc_rglt_typ_unit` | IP / ID 구분 |
| ID | `acc_rglt_val_id` | 차단 대상 ID |
| IP | `acc_rglt_val_ip` | 차단 대상 IP |
| 사유 | `reg_reas` | 차단 사유 |
| 차단일자 | `reg_ts` | — |

버튼: `등록`(차단 규칙 생성), `일괄 해제`(선택한 규칙 다중 해제)

| 동작 | API | 설명 | 명세 |
|------|-----|------|------|
| 목록 | `GET /v1/operation/acc/rglt` | params: `rglt_tgt_cd='M008USR'` | — |
| 등록 | `POST /v1/operation/acc/rglt` | 차단 규칙 생성 | — |
| 다중 해제 | `PUT /v1/operation/acc/rglt/multi` | `acc_rglt_id[]` + `acc_rglt_clr_tm` | — |

### Admin 접근 통제 (T060402)

경로: /service-manage/access-control/settings

컴포넌트: views/service/accessControl/AccessControlSettings.vue

User 접근 통제와 동일한 구조. `rglt_tgt_cd='M008ADM'`으로 관리자 대상.

부속 컴포넌트 `OutsideLinkageSetting` — 접근 통제 정책 설정(외부 연동 차단 정책).

| 동작 | API | 명세 |
|------|-----|------|
| 정책 상세 | `GET /v1/operation/acc/rglt/plcy/{id}` | — |
| 정책 수정 | `PUT /v1/operation/acc/rglt/plcy/{id}` | — |

## 외부 연동 (T0605)

외부 시스템과의 연동 설정. 각 메뉴가 독립된 설정 폼. disabled 조건은 각 메뉴 자체의 상태 플래그로 제어.

### AD 연동 (T060501)

경로: /service-manage/external-link/external-link-adLinkage

컴포넌트: views/service/ExternalLink/ExternalLinkAdLinkage.vue

Active Directory 연동 설정. disabled 제어: `adDisable` 플래그.

| 동작 | API | 명세 |
|------|-----|------|
| 연동 설정 조회 | `GET /v1/operation/outs/interfaces/usr` | [명세](../../api/operation/external/01-external.md) |
| 연동 설정 수정 | `PUT /v1/operation/outs/interfaces/usr` | [명세](../../api/operation/external/01-external.md) |
| AD 스크립트 조회 | `GET /v1/resource/adscript/detail/{tnt_id}` | [명세](../../api/resource/tenants/01-tenants.md) |
| AD 스크립트 수정 | `PUT /v1/resource/adscript/{id}` | [명세](../../api/resource/tenants/01-tenants.md) |
| AD 연결 테스트 | `POST /v1/legacy/ad/check` | — |
| OU 관리 | `POST /v1/legacy/cloud/ad/ous` (timeout 30초) | — |

### 네트워크 스토리지 연동 (T060502)

경로: /service-manage/external-link/external-link-networkStorage

컴포넌트: views/service/ExternalLink/ExternalLinkNetworkStorage.vue

NetApp NAS 연동 설정. disabled 제어: `nwDisable` 플래그.

| 필드 | 설명 | disabled 조건 |
|------|------|-------------|
| IP (`ip`) | NAS 서버 IP | `nwDisable` |
| Port (`port`) | 포트 | `nwDisable` |
| 인증 ID (`cert_id`) | 접속 ID | `nwDisable` |
| 비밀번호 (`passwd_cryptval`) | 접속 비밀번호 | `!tmp_network_reinput \|\| nwDisable` |
| 그룹 폴더 크기 (`grp_fold_cre_size`) | 기본 폴더 크기 | `nwDisable` |
| 설명 (`descp`) | 메모 | `nwDisable` |

| 동작 | API | 명세 |
|------|-----|------|
| 조회 | `GET /v1/operation/outs/interfaces/netapp` | [명세](../../api/operation/external/01-external.md) |
| 수정 | `PUT /v1/operation/outs/interfaces/netapp` | [명세](../../api/operation/external/01-external.md) |
