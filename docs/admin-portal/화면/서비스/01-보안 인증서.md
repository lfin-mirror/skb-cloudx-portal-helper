# 보안 인증서 (A0603)

SA 전용 (TA 메뉴에 없음). Kubernetes 환경의 TLS/SSL 인증서 관리 및 유효기간 알림.

## 2개 하위 메뉴

```
보안 인증서 (A0603)
  ├── 보안 인증서 (A060301) — 인증서 조회/업로드/다운로드
  └── 유효기간 알림 (A060302) — 만료 알림 이력 조회
```

---

## 보안 인증서 (A060301)

경로: /service-manage/security-certificate/histories

컴포넌트: views/service/securityCertificate/SecurityCertificateHistories.vue

인프라 구성 요소별 CA/서버 인증서 관리.

### 탭 구조

| 탭 | 대상 |
|----|------|
| all | 전체 |
| infra | INFRA |
| cloudpc | CloudPC |
| domain | Domain |
| spice | Spice (VDI 프로토콜) |

각 탭마다 CA 인증서 목록(조회만), 서버 인증서 목록(검색/페이지네이션). Spice/Domain 탭에서만 업로드/다운로드 버튼 표시.

### 테이블 컬럼

| 컬럼 | 필드 |
|------|------|
| 인증서 유형 | `ca_typ_cd` (M006PV: 사설 등) |
| 네임스페이스 | `namespace` |
| Secret 이름 | `secret_name` |
| 일련번호 | `serial_number` |
| 유효기간 | `valid_start` ~ `valid_end` |
| 버전 | `version` |
| 발급자 | `issuer` |
| 발급 대상 | `subject` |

### API

| 동작 | 경로 | 명세 |
|------|------|------|
| CA 목록 | `GET /v1/operation/secu/cert/list/{tab}/ca` | [명세](../../api/operation/cert/02-security-certificate.md) |
| 서버 목록 | `GET /v1/operation/secu/cert/list/{tab}/server` | [명세](../../api/operation/cert/02-security-certificate.md) |
| 다운로드 | `GET /v1/operation/secu/cert/download/{namespace}/{secret_name}` | [명세](../../api/operation/cert/02-security-certificate.md) |
| 업로드 | `POST /v1/operation/secu/cert/upload` | [명세](../../api/operation/cert/02-security-certificate.md) |

## 유효기간 알림 (A060302)

경로: /service-manage/security-certificate/expiration-date-histories

컴포넌트: views/service/securityCertificate/ExpirationDateHistories.vue

인증서 만료 임박/만료 알림 이력 조회. 읽기 전용. Excel 다운로드 지원.

### 검색 필터

| 필드 | 설명 |
|------|------|
| `infm_typ_cd` | 알림 유형 (M005 코드) |
| `scert_typ_cd` | 인증서 유형 (M006 코드) |

### 테이블 컬럼

| 컬럼 | 필드 |
|------|------|
| No. | `scert_infm_no` |
| 알림일시 | `infm_tm` |
| 알림유형 | `infm_typ_cd_nm` |
| 인증서유형 | `scert_typ_cd_nm` |
| 호스트명 | `host_nm` |
| 유효기간 | `vlid_stt_dt` ~ `vlid_end_dt` |
| 인증서 버전 | `scert_ver` |
| 발급자 | `isur_nm` |

### API

| 동작 | 경로 | 명세 |
|------|------|------|
| 조회 | `GET /v1/operation/secu/cert/infm` | [명세](../../api/operation/cert/02-security-certificate.md) |
