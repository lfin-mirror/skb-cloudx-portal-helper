# 팝업 (T0407)

user-portal 로그인/메인 화면에 표시되는 팝업 공지 관리.

- 경로: `/portal/popup`
- 컴포넌트: `views/portal/Popup.vue`
- 등록/수정: `views/portal/PopupDetail.vue`

## 기능

단일 팝업 관리 (한 번에 하나만 설정 가능). 팝업 표시 여부, 필수 확인 여부, 표시 기간 설정.

## API

| 동작 | 메서드 | 경로 | 명세 |
|------|--------|------|------|
| 조회 | GET | `/v1/system/popup` | [명세](../../api/system/popup/01-popup.md) |
| 등록 | POST | `/v1/system/popup` | [명세](../../api/system/popup/01-popup.md) |
| 수정 | PUT | `/v1/system/popup` | [명세](../../api/system/popup/01-popup.md) |

## 관리 항목

| 항목 | 필드 | 설명 |
|------|------|------|
| 표시 여부 | `show_yn` | `Y`: 표시 / `N`: 숨김 |
| 필수 확인 | `check_yn` | `Y`: 사용자가 반드시 확인해야 함 |
| 표시 시작 | `show_stt_dtm_str` | 표시 시작 일시 |
| 표시 종료 | `show_end_dtm_str` | 표시 종료 일시 |
| 내용 | `cont` | HTML 에디터 (WYSIWYG) |
| 첨부 이미지 | `wri_att_file_l` | 팝업 내 이미지 |

## user-portal 연관

팝업이 없으면 "등록" 버튼, 있으면 조회/수정 모드. 수정으로 `show_yn='N'` 설정 시 비활성화 (삭제 기능 없음).

user-portal에서는 로그인 또는 메인 페이지 진입 시 팝업 조회. `check_yn='Y'`면 사용자가 확인 버튼을 눌러야 닫힘.
