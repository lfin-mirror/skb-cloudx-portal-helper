# 사용자 UI (T0401)

TA 전용 (SA 메뉴에 없음). user-portal의 포털 UI 커스터마이징 설정.

경로: /portal/user

컴포넌트: views/portal/ui/UserPortal.vue / UserPortalDetail.vue

user-portal의 로그인 화면, 메인 화면, Footer에 표시되는 이미지/텍스트를 테넌트별로 설정.

## 관리 항목

| 항목 | 필드 | 설명 |
|------|------|------|
| 로그인 배너 1~3 | `lin_img_1/2/3_basic_yn` | 최대 3개 배너 이미지 |
| 로그인 배경 | `lin_bg_yn` / `lin_bg_stor_path` | 사용/미사용 |
| 포털 로고 | `ptal_img_basic_yn` / `ptal_img_stor_path` | user-portal 상단 로고 |
| 포털 배경 | `ptal_bg_basic_yn` / `ptal_bg_stor_path` | user-portal 메인 배경 |
| Favicon | `fvcn_yn` / `fvcn_stor_path` | 브라우저 탭 아이콘 |
| 저작권 | `copyright_txt` | Footer 저작권 문구 |
| 이용문의 | `inquiry_txt` | Footer 이용문의 텍스트 |
| 연락처 | `contact_txt` | Footer 연락처 텍스트 |

## user-portal에서의 적용

```
user-portal 접속 → /:tenant 경로에서 tnt_url_id 결정
  → GET /v1/nauth/system/portals/ui/{tntUrlId}/public
  → Vuex state + localStorage 저장
  → user-portal의 Layout, Login, Footer 컴포넌트에 반영
```

멀티테넌트 환경에서 각 테넌트가 자체 브랜딩 가능.

## 기능 개선 요청 설정 (T0408)

UserPortal.vue 내부에 `UserPortalImproves` 컴포넌트 포함. user-portal의 "기능 개선 요청" 메뉴 표시 여부 토글.

| 동작 | 경로 |
|------|------|
| 사용 여부 조회 | `GET /v1/system/improves/menu/usg` |
| 사용 여부 변경 | `PUT /v1/system/improves/menu/usg` (`usg_yn`: Y/N) |
