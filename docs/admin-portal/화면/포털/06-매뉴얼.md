# 매뉴얼 (T0405)

user-portal에서 다운로드 가능한 매뉴얼 파일 관리.

- 경로: `/portal/manual`
- 컴포넌트: `views/portal/Manual.vue`
- 상세/수정: `views/portal/ManualDetail.vue`

## 기능

매뉴얼 파일 목록 조회 및 수정. 초기 설정 시 12개 템플릿 항목 자동 생성. 삭제 불가.

## API

| 동작 | 메서드 | 경로 | 명세 |
|------|--------|------|------|
| 목록 조회 | GET | `/v1/system/portals/guides` | [명세](../../api/system/portals/02-portals-guides.md) |
| 초기 등록 | POST | `/v1/system/portals/guides/create` | [명세](../../api/system/portals/02-portals-guides.md) |
| 상세 조회 | GET | `/v1/system/portals/guides/{id}` | [명세](../../api/system/portals/02-portals-guides.md) |
| 수정 | PUT | `/v1/system/portals/guides/{id}` | [명세](../../api/system/portals/02-portals-guides.md) |

목록 조회 시 `ui_file_div_cd = 'A009L'`인 항목만 필터링하여 표시. 12개 미만이면 "초기등록" 버튼 노출.

## 테이블 컬럼

| 컬럼 | 필드 | 정렬 |
|------|------|------|
| 파일 분류 | `ui_file_div_nm` | O |
| 파일명 | `ui_file_nm` | O |
| 생성자 | `reg_conn_id` | O |
| 생성일시 | `reg_ts` | O |
| 수정자 | `mod_conn_id` | O |
| 수정일시 | `mod_ts` | O |

## user-portal 연관

admin-portal에서 등록한 매뉴얼 → user-portal `support/Manual.vue`에서 다운로드.

| admin-portal | user-portal |
|-------------|-------------|
| 매뉴얼 파일 업로드/수정 | PDF 다운로드 |
| 12개 분류 항목 관리 | PC Client/Mobile Client/Thin Client 탭 |
| `ui_file_div_cd = 'A009L'` | 동일 코드로 조회 |

동일 API 엔드포인트(`/v1/system/portals/guides`) 사용.
