# 공지사항 (T0402)

user-portal에 표시되는 공지사항 CRUD 관리.

- 경로: `/portal/notice`
- 컴포넌트: `views/portal/Notice.vue`
- 등록: `views/portal/NoticeCreate.vue`
- 상세/수정: `views/portal/NoticeDetail.vue`

## 기능

공지사항 목록 조회, 등록, 수정, 삭제. 대표 공지(`rep_yn`) 설정으로 user-portal 홈 화면에 우선 노출 가능.

## API

| 동작 | 메서드 | 경로 | 명세 |
|------|--------|------|------|
| 목록 조회 | GET | `/v1/system/notices` | [명세](../../api/system/notices/01-notices.md) |
| 등록 | POST | `/v1/system/notices` | [명세](../../api/system/notices/01-notices.md) |
| 상세 조회 | GET | `/v1/system/notices/{id}` | [명세](../../api/system/notices/01-notices.md) |
| 수정 | PUT | `/v1/system/notices/{id}` | [명세](../../api/system/notices/01-notices.md) |
| 삭제 | DELETE | `/v1/system/notices/{id}` | [명세](../../api/system/notices/01-notices.md) |

## 검색 조건

| 파라미터 | 설명 |
|---------|------|
| `type` | 검색 타입 (title / content / reg_conn_id) |
| `value` | 검색어 |
| `rep_yn_nm` | 대표 공지 여부 (Y / N) |
| `page`, `limit` | 페이지네이션 |

## 테이블 컬럼

| 컬럼 | 필드 | 정렬 |
|------|------|------|
| 대표 공지 | `rep_yn_nm` | O |
| 제목 | `title` (unescape 처리) | O |
| 생성자 | `reg_conn_id` | O |
| 생성일시 | `reg_ts` (날짜 포맷) | O |

## user-portal 연관

admin-portal에서 등록한 공지 → user-portal `support/NoticeList.vue`에서 조회.

| admin-portal | user-portal |
|-------------|-------------|
| 공지 등록/수정/삭제 | 공지 목록 조회 + 상세 조회 |
| 대표 공지 설정 (`rep_yn='U016REP'`) | 홈 화면 롤링 공지에 우선 노출 |
| 검색: 제목/내용/작성자 | 검색: 제목/내용 |
| 페이지네이션 (data-tables) | 50건씩 "더보기" 방식 |

동일 API 엔드포인트(`/v1/system/notices`) 사용. user-portal은 `noti_tgt_grp_id` 파라미터로 테넌트 필터링.
