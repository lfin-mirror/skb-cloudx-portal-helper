# 1:1문의 (T0404)

user-portal 사용자가 등록한 문의에 대해 조회·답변하는 관리 화면.

- 경로: `/portal/man-to-man`
- 컴포넌트: `views/portal/ManToMan.vue`
- 상세/답변: `views/portal/ManToManDetail.vue`

## 기능

사용자 문의(QNA) 목록 조회 및 답변 등록/수정. 관리자는 문의를 생성하거나 삭제할 수 없음 — user-portal에서만 등록 가능.

## API

| 동작 | 메서드 | 경로 | 명세 |
|------|--------|------|------|
| 목록 조회 | GET | `/v1/system/qnas` | [명세](../../api/system/qnas/01-qnas.md) |
| 상세 조회 | GET | `/v1/system/qnas/{id}` | [명세](../../api/system/qnas/01-qnas.md) |
| 답변 등록/수정 | PUT | `/v1/system/qnas/{id}` | [명세](../../api/system/qnas/01-qnas.md) |

## 검색 조건

7가지 검색 타입:

| 타입 코드 | 설명 |
|----------|------|
| `T` | 제목 검색 |
| `Q_ID` | 작성자 ID (포함) |
| `Q_ID_EQ` | 작성자 ID (정확히 일치) |
| `Q` | 작성자명 (포함) |
| `A_ID` | 응답자 ID (포함) |
| `A_ID_EQ` | 응답자 ID (정확히 일치) |
| `A` | 응답자명 (포함) |

## 테이블 컬럼

| 컬럼 | 필드 | 정렬 |
|------|------|------|
| 문의 타입 | `adv_typ_cd_nm` (A003 코드) | O |
| 제목 | `title` (unescape 처리) | O |
| 답변 상태 | `adv_pgrs_sts_cd_nm` | O |
| 사용자 그룹 | `usr_grp_nm` | O |
| 문의자 ID | `adv_req_acct_conn_id` | O |
| 문의자명 | `adv_req_acct_nm` | O |
| 문의일시 | `reg_ts` (날짜 포맷) | O |

## user-portal 연관

| admin-portal | user-portal |
|-------------|-------------|
| 문의 조회 + 답변 등록/수정 | 문의 등록 + 조회 + 수정/삭제 (답변대기 시만) |
| 전체 사용자 문의 조회 가능 | 본인 문의만 조회 |
| 7가지 검색 타입 | 없음 |
| 답변 상태 추적 | 답변대기/답변완료 건수 표시 |

user-portal `support/ContactList.vue`에서 `POST /v1/system/qnas`로 문의 등록. admin-portal에서 `PUT`으로 답변. 동일 API 엔드포인트 공유.
