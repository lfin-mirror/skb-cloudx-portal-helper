# 관리자 UI (A0401)

SA 전용 (TA 메뉴에 없음). admin-portal 자체의 포털 UI 커스터마이징 설정.

경로: /portal/admin

컴포넌트: views/portal/ui/AdminPortal.vue / AdminPortalDetail.vue

admin-portal의 로그인 화면, Navbar, Footer에 표시되는 이미지/텍스트 설정을 조회·수정.

## 관리 항목

| 항목 | 필드 | 설명 |
|------|------|------|
| 로그인 로고 | `lin_img_1_basic_yn` / `lin_img_1_stor_path` | 기본(`Y`) 또는 직접 등록(`N`) |
| 로그인 배경 | `lin_bg_yn` / `lin_bg_stor_path` | 사용(`Y`) / 미사용(`N`) |
| 포털 로고 | `ptal_img_basic_yn` / `ptal_img_stor_path` | Navbar 좌측 로고 |
| 포털 배경 | `ptal_bg_basic_yn` / `ptal_bg_stor_path` | 메인 배경 이미지 |
| 로그인 타이틀 | `ptal_lin_title` | 로그인 화면 제목 (기본: `'Cloud X Service Portal'`) |
| 저작권 | `copyright_txt` | Footer 저작권 문구 |
| 이용문의 | `inquiry_txt` | Footer 이용문의 텍스트 |
| 연락처 | `contact_txt` | Footer 연락처 텍스트 |

## 사용자 UI (T0401)와의 차이

| 항목 | SA 관리자 UI (A0401) | TA 사용자 UI (T0401) |
|------|-----------|-----------|
| 대상 | admin-portal 자체 | user-portal |
| API 경로 | `/v1/system/portals/ui/ADMIN` | `/v1/system/portals/ui/{tntId}` |
| 공개 API | `/v1/nauth/.../ADMIN/public` | `/v1/nauth/.../{tntUrlId}/public` |
| Favicon | 미지원 | 지원 (`fvcn_yn`, `fvcn_stor_path`) |
| 로그인 배너 | 1개 이미지 | 3개 이미지 (`lin_img_1/2/3`) |
| 기능 개선 요청 설정 | 없음 | UserPortalImproves 컴포넌트 포함 |

## 설정 반영 경로

```
AdminPortalDetail에서 수정 → POST /v1/system/portals/ui/ADMIN
  → 페이지 새로고침 (this.$router.go())
  → permission.js의 router.beforeEach
  → store.dispatch('setImages') (images.isLoad === false일 때)
  → GET /v1/nauth/system/portals/ui/ADMIN/public
  → Vuex state 업데이트
```
