# 사용자 그룹 (T0201)

경로: /user/group

컴포넌트: views/userInfo/Group.vue

상세: views/userInfo/GroupDetail.vue

TA 전용 (SA 메뉴에 없음). `isSuperAdmin` 분기 없음.

사용자를 부서/팀 단위로 묶는 그룹 관리. 그룹별 인증 정책, 근태 관리, 공유 폴더 등을 설정.

## 연관 관계

```
[사용자 그룹] (T0201)
     │
     │  그룹에 소속된 사용자
     ▼
[사용자] (T0202)
     │
     │  사용자에게 VM 할당
     ▼
[가상 PC 그룹] (T0701) → 풀 → VM
     │
     │  자동 할당 맵핑 설정 시
     ▼
[자동 할당 맵핑] (T0707)   사용자 그룹 ↔ Pool 매핑
```

## UI 레이아웃

좌측 폼(8칸) + 우측 리스트(16칸) 2단 구조. 행 클릭 시 좌측에 상세 폼 표시.

### 필터

| 필터 | 필드 | 설명 |
|------|------|------|
| 그룹명 | `usr_grp_nm` | 검색 |
| 인증정책 대상코드 | `cert_plcy_tgt_cd` | 드롭다운 (코드: U003) |

### 테이블 컬럼

| 컬럼 | 필드 | 설명 |
|------|------|------|
| 그룹명 | `usr_grp_nm` | — |

| 가상PC 수 | `vm_as_cnt` | — |
| 근태 관리 설정 | `work_mng_use_yn` | Y/N |
| 클라이언트 업데이트 여부 | `client_update_yn` | Y/N |

### 버튼

| 버튼 | 설명 |
|------|------|
| 신규 | 좌측에 신규 등록 폼 표시 |
| 할당 | `VirtualPcAssignment` 모달 — 그룹 단위 VM 일괄 할당 (`pageType='userGroup'`) |
| 기간변경 | `GroupPeriod` 모달 — 선택된 그룹의 사용자 계정 유효기간 일괄 변경 |
| 파일업로드 | `GroupUpload` 모달 — CSV 파일로 사용자 일괄 등록 (`bizCode: 'A002R'`) |
| AD사용자그룹동기화 | `POST /v1/legacy/customer/ad/group/sync/{tnt_id}` — AD 연동 시에만 표시 (`adChk` 플래그) |
| CSV다운로드 | 그룹 목록 JSON 다운로드 |
