# 사용자 (T0202)

경로: /user/index

컴포넌트: views/userInfo/userManage.vue

상세: views/userInfo/userManageDetail.vue

TA 전용 (SA 메뉴에 없음). `isSuperAdmin` 분기 없음.

테넌트 소속 사용자 계정 관리. 사용자 등록/수정/삭제, VM 할당, 개인 디스크 관리.

## UI 레이아웃

좌측 폼(8칸) + 우측 리스트(16칸) 2단 구조. 행 클릭 시 좌측에 상세 폼 표시.

리스트의 각 행에 **Expand Row** — 펼치면 해당 사용자의 가상 PC / 개인 디스크 목록 표시.

### Expand Row

행을 펼치면 사용자의 VM 목록 표시:

| 상태 | 표시 내용 |
|------|----------|
| 재할당 준비 / 할당 준비 / 할당 실패 | 상태 메시지만 표시 |
| `isVmPowerOnCondition` = TRUE | 유저접속상태(`usr_vm_conn_sts_cd_nm`) / 테넌트 / VM명 |
| `isVmPowerOnCondition` = FALSE | 공용PC예약 / 테넌트 / VM명 |

Expand Row 내 버튼:
- `개별 설정` — 개별 VM 상세 설정 (disabled: 재할당 상태일 때)
- `관리` — 개인 디스크 관리 (개인 디스크 존재 시)
- `가상PC 할당` — VM 할당 (disabled: `acct_sts_cd !== 'U013NR'`, 정상 상태에서만 가능)

### 버튼

| 버튼 | 설명 |
|------|------|
| 신규 | 좌측에 신규 등록 폼 표시 |
| 할당 | `VirtualPcAssignment` 모달 — 사용자 단위 VM 할당 (`pageType='user'`) |
| 삭제 | 만료된 사용자 일괄 삭제 (`DELETE /v1/user/accounts`, body에 만료 사용자 목록) |
| CSV업로드 | `userUpload` 모달 — CSV 파일로 사용자 일괄 등록 |
| 인증정책일괄변경 | `CertificationPolicySettingForModal` 모달 — 선택한 사용자들의 인증 정책 일괄 변경 |
| 상태일괄변경 | `StatusSettingForModal` 모달 — 선택한 사용자들의 계정 상태 일괄 변경 |
| AD사용자목록호출 | AD 연동 시에만 표시 |
| CSV다운로드 | 사용자 목록 다운로드 |
