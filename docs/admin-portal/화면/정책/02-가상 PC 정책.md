# 가상 PC 정책 (A0503, T0503)

- 경로: `/policy/virtual-pc-auth-policy`
- 컴포넌트: `VirtualPcAuthPolicySupadm.vue`

VM 접속 시 적용되는 보안/동작 정책. 상세 정책(A0506) 하위에서 개별 생성한 정책들을 **하나로 묶어서 VM에 적용하는 허브**.

## UI 레이아웃

폼 기반 단일 화면. 아래 10개 설정 섹션이 상하로 나열. 하단에 `저장` 버튼 1개.

## 설정 항목

### 1. 가상 PC 보안 정책 설정

`secu_plcy_nm` (정책명, 읽기 전용) + `CommonSelectList` 컴포넌트로 구성.

VM 접속 시 적용할 **보안 기능 8개 항목**을 "기본 정책" 컬럼과 "사용자정의 네트워크 정책" 컬럼으로 나누어 각각 허용/제한 설정.

#### 보안 항목 테이블

`CommonSelectList`가 렌더링하는 테이블. 행(row)은 보안 항목, 열(column)은 네트워크 구분.

| 행 (보안 항목) | `cdKey` | 코드 그룹 | 선택지 |
|---------------|---------|----------|--------|
| 클립보드 공유 | `clb_shar_auth_cd` | U009 | 허용/제한/양방향 등 |
| 파일 전송(Drag & Drop) | `drag_drop_auth_cd` | U010 | 허용/제한 등 |
| 로컬 USB 연결 | `usb_plcy_id` | USB 정책 목록 | USB 정책(A050609) 선택 |
| URL Redirection | `url_rdrt_auth_cd` | U012 | 허용/제한 |
| 로컬 프린터 연결 | `prt_conn_auth_cd` | U007 | 허용/제한 |
| 다중 모니터 | `mult_mtor_auth_cd` | U026 | 허용/제한 |
| 화면캡쳐 여부 | `scr_capture_yn` | — | Y(허용)/N(제한) |
| 접근 통제(보안 그룹) | `sg_grp_id` | 보안 그룹 목록 | 보안 그룹(A050611) 선택. **SA: forceDisabled** |

주석 처리된 항목:
- `fold_shar_auth_cd` (로컬 PC 폴더 공유) — IAADEV-4744에서 삭제
- `usb_conn_auth_cd` (USB 연결) — `usb_plcy_id`로 대체

#### 컬럼 구조: 기본 정책 vs 사용자정의 네트워크 정책

| 컬럼 | 키 | 설명 |
|------|----|------|
| 기본 정책 (`tot`) | `form.tot[cdKey]` | 전체 네트워크 대상 기본 설정 |
| 사용자정의 네트워크 정책 (`ext`) | `form.ext[cdKey]` | 예외 네트워크 그룹에 속한 IP 대상 설정 |

저장 시 `pcly_cert` 배열로 `[tot, ext]` 두 벌이 서버에 전송 — 각각 `secu_plcy_auth_id`를 가짐.

#### 예외 네트워크 그룹 연결

- `예외 네트워크 설정` 버튼 → `ExcerptNetworkSettingModal` 모달 → 예외 네트워크(A050601)에서 생성한 그룹 선택
- 선택 시 `exc_nw_grp_id` 세팅 + 그룹명(`exc_nw_grp_nm`)을 disabled input에 표시
- `삭제` 버튼 → 예외 네트워크 연결 해제, "사용자정의 네트워크 정책" 컬럼 전체 disabled
- **SA에서는 두 버튼 모두 비노출** (`isUse: !isSuperAdmin`)

#### 2차 인증과의 관계

`nw_n2nd_cert_yn` 키가 테이블 행에 포함될 때 (인증 정책에서 사용), 기본/사용자정의 조합에 따라 2차 인증 범위가 결정:

| 기본 정책 | 사용자정의 네트워크 정책 | 결과 |
|-----------|----------------------|------|
| 사용 | 미사용 | 사용자정의 네트워크를 **제외한** 모든 네트워크에서 2차 인증 |
| 미사용 | 사용 | 사용자정의 네트워크 IP에서**만** 2차 인증 |
| 사용 | 사용 | 무조건 2차 인증 |

#### disabled 조건 정리

```
예외 네트워크 미설정 → "사용자정의 네트워크 정책" 컬럼 전체 disabled (isExtDisabled = true)
SA 접속             → 예외 네트워크 버튼 비노출 + sg_grp_id 행 forceDisabled
nw_n2nd_cert_yn='N' → 해당 행 하위의 SMS 인증 방식 항목 disabled
```

### 2. 자동 해상도 설정

| 필드 | 값 |
|------|-----|
| `auto_resol_usg_yn` | Y/N 라디오 |

VM 접속 시 클라이언트 해상도에 맞춰 자동 조정 여부.

### 3. 가상 PC 자동 로그인

| 필드 | 값 |
|------|-----|
| `vpc_auto_login_yn` | Y/N 라디오 |

VM 접속 시 OS 로그인 자동 처리 여부.

### 4. 워터마크 설정

| 필드 | 값 |
|------|-----|
| `wmk_usg_yn` | Y/N 라디오 |

`wmk_usg_yn == 'Y'`일 때 추가 설정 노출:

- **표시 항목**: 순서 테이블에서 워터마크에 표시할 항목 선택 (사용자명, IP, 날짜 등). 순서 변경 가능.
- **상세 설정**: `WaterMarkControl` 컴포넌트로 폰트, 크기, 투명도, 위치, 각도 조절. `상세 설정` 버튼 클릭 시 모달.
- **항목 초기화**: `항목 초기화` 버튼으로 선택 초기화.

VM 화면 위에 반투명 텍스트를 오버레이하여 화면 캡처/유출 방지.

### 5. 클라이언트 세션 만료 정책

| 필드 | 값 |
|------|-----|
| `viwr_expr_usg_yn` | Y/N 라디오 |
| `viwr_expr_mi_cnt` | 분 단위 입력 (기본값 60). `viwr_expr_usg_yn == 'Y'`일 때 활성 |

VM 뷰어(클라이언트)가 일정 시간 미사용 시 자동 세션 종료.

### 6. 접근 차단 정책 설정 (SA: disabled)

| 필드 | 값 |
|------|-----|
| `acc_blck_plcy_id` | 드롭다운 — 접근 차단 정책(A050602)에서 생성한 정책 선택 |

선택 시 하단에 해당 정책의 서비스별 차단 규칙 테이블 표시:

| 컬럼 | 필드 | 설명 |
|------|------|------|
| 서비스명 | `acc_blck_serv_nm` | 차단 대상 서비스 |
| IP | `ip` | 서비스 IP |
| Port | `port` | 서비스 포트 |
| 전체 네트워크 | `tot_net_blck_yn` | 전체 네트워크에서의 차단 여부 (Y/N 드롭다운) |
| 예외 네트워크 | `exc_net_blck_yn` | 예외 네트워크에서의 차단 여부 (Y/N 드롭다운) |

SA에서는 드롭다운 **disabled**. 보안 그룹 목록(`sgGrpList`)은 `[{ label: 'default', value: null }]`로 하드코딩 — SA에서는 기본 정책이 설정된 것처럼 보이되 변경 불가.

## SA vs TA 차이

SA는 `VirtualPcAuthPolicySupadm.vue` (단일 폼 편집), TA는 `VirtualPcAuthPolicy.vue` (목록 + 상세 폼). 파일 자체가 분리.


### 목록 화면

| 항목 | 세부 항목 | SA (`*Supadm.vue`) | TA (`*.vue`) |
|------|----------|-------------------|-------------|
| **UI 구조** | 화면 형태 | 폼 1개 직접 편집 (목록 없음) | 목록 + 상세 폼 (다중 정책) |
| **목록** | `secu_plcy_tgt_cd` 파라미터 | — (목록 없음) | `U006T` (TA 정책만) |
| | 정책명 컬럼 (`secu_plcy_nm`) | — | 표시 |
| | 정책 타입 컬럼 (`secu_plcy_tgt_cd_nm`) | — | 표시 |
| | Group 수 (`group_cnt`) | — | **TA 전용** (클릭 시 모달) |
| | Pool 수 (`pool_cnt`) | — | **TA 전용** (클릭 시 모달) |
| | VM 수 (`vm_cnt`) | — | **TA 전용** (클릭 시 모달) |
| **등록** | 등록 버튼 + 생성 가이드 | **비노출** | 노출 (`v-if="!isSuperAdmin"`) |
| **SA 정책 필터링** | SA 정책 행 클릭 | — | `secu_plcy_tgt_cd !== 'U006S'` 행만 클릭 가능 |
