# 인증 정책 (A0501/A0502, T0501/T0502)

사용자 인증 정책(A0501)과 관리자 인증 정책(A0502)을 통합 정리. 두 메뉴는 대상만 다르고 구조가 동일.

## 두 메뉴의 차이

| 항목 | 사용자 인증 정책 (A0501) | 관리자 인증 정책 (A0502) |
|------|------------------------|------------------------|
| 경로 | `/policy/user-auth-policy` | `/policy/admin-auth-policy` |
| SA 컴포넌트 | `UserAuthPolicySupadm.vue` | `AdminAuthPolicySupadm.vue` |
| TA 컴포넌트 | `UserAuthPolicy.vue` | `AdminAuthPolicy.vue` |
| API | `GET /v1/operation/cert/n2nd/ptalType/A007USR` | `GET /v1/operation/cert/n2nd/ptalType/A007ADM` |
| 대상 코드 | `cert_plcy_tgt_cd`: U003S(SA용) / U003B(TA용) | 동일 |
| 정책 적용 대상 | 사용자 포털 로그인 | 관리자 포털 로그인 |

SA 화면에서는 **정책 템플릿을 정의**하는 역할. TA가 실제로 사용자/관리자에게 정책을 적용.

## 폼 필드

| 화면 라벨 (i18n) | 필드 | 타입 | disabled 조건 |
|-----------------|------|------|-------------|
| 정책명 (`page.policy.label.userPolicyName`) | `cert_plcy_nm` | 텍스트 (readonly) | — |
| 로그인 제한 사용 여부 (`page.policy.label.isLoginPolicyUse`) | `lin_rstt_usg_yn` | 라디오 (C: 횟수 / T: 시간 / N: 미사용) | — |
| | ↓ **lin_rstt_usg_yn = 'C'일 때만 활성** | | |
| 로그인 실패 허용 횟수 (`page.policy.label.loginLimitStandard`) | `lin_fail_alow_cnt` | InputNumber (1~99999) | `lin_rstt_usg_yn != 'C'` |
| | ↓ **lin_rstt_usg_yn = 'T'일 때만 활성** | | |
| 로그인 제한 시간 (`page.policy.label.loginLimitStandard`) | `lin_rstt_mi_cnt` | InputNumber (분, 1~99999) | `lin_rstt_usg_yn != 'T'` |
| | ↓ **lin_rstt_usg_yn ≠ 'N'일 때만 활성** | | |
| 자동 로그아웃 기준 시간 | `sess_expr_mi_cnt` | InputNumber (분) | `lin_rstt_usg_yn == 'N'` |
| 인증서 설정 | 2차 인증 관련 | 예외 네트워크 그룹 선택 포함 | — |

### 로그인 제한 방식

```
C (횟수 기반): N회 연속 실패 → 계정 잠금
T (시간 기반): N분 내 실패 누적 → 계정 잠금
N (미사용):   제한 없음
```

## SA vs TA 차이

| 항목 | 세부 항목 | SA | TA |
|------|----------|----|----|
| **대상 코드** | `cert_plcy_tgt_cd` | `U003S` | `U003B` |
| **UI 구조** | — | 폼 1개 직접 편집 (`*Supadm.vue`) | 목록 + 상세 폼 (`*.vue`) |
| **역할** | — | 기본 정책 템플릿 정의 | 테넌트별 정책 커스터마이징 |
| **등록 버튼** | — | 비노출 | 노출 (`v-if="!isSuperAdmin"`) |
| **삭제 버튼** | — | 비노출 | 노출 (`v-if="id && !isSuperAdmin"`) |
| **정책 가져오기** | — | 비노출 | 노출 (`v-if="!id && !isSuperAdmin"`) — SA 템플릿 복사 |
| **사용 현황** | 그룹 수 / 사용자 수 | 비노출 | 표시 (클릭 시 상세 모달) |
| **SA 정책 필터링** | 목록 행 클릭 | — | `cert_plcy_tgt_cd !== 'U003S'` 행만 클릭 가능 |

## 버튼 동작

| 버튼 | SA | TA | 클릭 시 동작 |
|------|----|----|------------|
| **저장** | 노출 | 노출 | `PUT /v1/operation/cert/n2nd/{id}` |
| **등록** | 비노출 | 노출 | 빈 폼 표시 → 저장 시 `POST` |

## 가상 PC 정책과의 관계

```
인증 정책 (A0501/A0502)     → 포털 로그인 시 2차 인증 제어
가상 PC 정책 (A0503)        → VM 접속 시 2차 인증 + 보안 설정 제어
                               └── 접근 차단 정책 참조
                               └── USB 정책 참조
```

인증 정책은 "포털에 로그인할 때" 적용, 가상 PC 정책은 "VM에 접속할 때" 적용. 둘 다 2차 인증 설정을 포함하지만 적용 시점이 다름.
