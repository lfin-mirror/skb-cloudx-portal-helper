# 정책 메뉴 개요

## SA와 TA의 역할 분담

SA는 `*Supadm.vue`(전역 템플릿 정의), TA는 `*.vue`(테넌트별 정책 생성/관리). 파일 자체가 분리된 구조.

```
[SA] *Supadm.vue — 전역 템플릿 정의
  → tgt_cd: U003S / U006S / U023S
  → 등록 버튼 없음 (단일 정책 편집)
  → 기본 정책 값 설정

[TA] *.vue — 테넌트별 정책 생성/관리
  → tgt_cd: U003B / U006T / U023T
  → 등록 버튼 있음 (여러 정책 생성 가능)
  → "정책 가져오기" 로 SA 템플릿 복사 가능
  → 삭제 가능
  → 사용 현황 (그룹/풀/VM 수) 확인 가능
  → SA 정책(tgt_cd=*S)은 목록에 보이지만 클릭 불가
```

## 하위 메뉴 구조

```
정책
├── 인증 정책 (A0501/T0501, A0502/T0502)   → 01-인증 정책.md
├── 가상 PC 정책 (A0503/T0503)              → 02-가상 PC 정책.md
├── 백업/스냅샷 정책 (A0505/T0505)           → 03-백업 스냅샷 정책.md
└── 상세 정책 (A0506/T0506)
    ├── 예외 네트워크 / 접근 차단              → 04-네트워크 보안 정책.md
    ├── 블랙/화이트리스트 / URL / 미라이선스    → 05-SW 제어 정책.md
    ├── USB 정책 4종                          → 06-USB 정책.md
    └── Metadata                              → 07-Metadata.md
```

## 상세 정책 — SA vs TA 메뉴 접근 차이

SA 상세 정책에는 **Metadata + USB 정책 4종만** 존재. 나머지는 **TA에만** 메뉴가 있음.

| 메뉴 | SA 메뉴ID | TA 메뉴ID | SA 접근 | TA 접근 |
|------|----------|----------|--------|--------|
| 예외 네트워크 | — | T050601 | **메뉴 없음** | CRUD |
| 접근 차단 | — | T050602 | **메뉴 없음** | CRUD |
| 블랙리스트 | — | T050604 | **메뉴 없음** | CRUD |
| 화이트리스트 | — | T050605 | **메뉴 없음** | CRUD |
| URL Redirection | — | T050606 | **메뉴 없음** | CRUD |
| 비인가 프로세스 | — | T050607 | **메뉴 없음** | CRUD |
| Metadata | A050608 | T050608 | 조회/수정 | CRUD (파일 분리) |
| USB 정책 | A05060901 | T05060901 | CRUD | CRUD |
| USB 유형 | A05060902 | T05060902 | CRUD | CRUD |
| USB 제조사 | A05060903 | T05060903 | CRUD | CRUD |
| USB 모델 | A05060904 | T05060904 | CRUD | CRUD |
| 가상PC 네이밍 정책 | — | T050610 | **메뉴 없음** | 라우트 /policy/policy-manage/virtual-pc-naming-policy 활성. VirtualPcNamingPolicy.vue 존재 |

## 상세 정책과 가상 PC 정책의 연결 관계

```
[상세 정책에서 생성]                         [가상 PC 정책에서 선택]
예외 네트워크 (T050601) ──────────────→ [1] 보안 정책 설정의 예외 네트워크 그룹
접근 차단 (T050602) ──────────────────→ [6] 접근 차단 정책 드롭다운
URL Redirection (T050606) ────────────→ [7] URL Redirection 예외 드롭다운
비인가 프로세스 (T050607) ────────────→ [8] 미라이선스 정책 드롭다운
Metadata (T050608) ───────────────────→ [9] Metadata 정책 드롭다운
USB 정책 (T05060901) ─────────────────→ [1] 보안 항목 테이블의 USB 정책 행
```
