# 가상 PC 그룹 (T0701)

경로: /virtual-pc/virtual-pc-create

컴포넌트: views/virtualPc/VirtualPcCreateList.vue

상세: views/virtualPc/VirtualPcCreateDetail.vue

풀 상세: views/virtualPc/VirtualPcPoolDetail.vue

TA 전용 (SA 메뉴에 없음). `isSuperAdmin` 분기 없음.

VM 프로비저닝의 최상위 단위. 그룹 → 풀(Pool) → VM 계층 구조.

## 계층 구조

```
가상 PC 그룹
  │  템플릿, 네트워크, 가상 PC 정책, 백업/스냅샷 정책 설정
  │
  ├── 풀 A (전용 PC, V001DED)
  │     Zone, 서브넷, 볼륨 타입, IP 할당 방식, 전원 정책 등 설정
  │     ├── VM-001 → 사용자 A (고정)
  │     └── VM-002 → 사용자 B (고정)
  │
  └── 풀 B (공용 PC, V001POO)
        Zone, 서브넷, 볼륨 타입, 대기 자원, 초기화 정책 등 설정
        ├── VM-003 → (대기 중, 풀에서 자동 배정)
        └── VM-004 → (대기 중)
```

전용/공용 PC 개념: [docs/term/03-virtual-pc-types.md](../../../term/03-virtual-pc-types.md) 참고.

## 연관 메뉴

```
[템플릿] (T0901)          Flavor + Image = 설계도
[가상 PC 정책] (T0503)    보안/접근 차단/USB/워터마크 등
[백업/스냅샷 정책] (T0505) 자동 백업/스냅샷 스케줄
[네트워크] (T1004)         가상 네트워크/서브넷
[스토리지] (T1005)         볼륨 타입
     │
     │  위 리소스들을 조합
     ▼
[가상 PC 그룹] (T0701)    그룹 = 위 설정 묶음
     │
     │  그룹 안에 풀 생성
     ▼
[가상 PC] (T0702)         풀에 할당된 개별 VM 목록
```

## 그룹 상세 (VirtualPcCreateDetail)

그룹 레벨에서 설정하는 항목. 하위 풀 전체에 공통 적용.

### 설정 항목

| 항목 | 필드 | 설명 |
|------|------|------|
| 가상 PC 그룹명 (`vm_grp_nm`) | 입력 | 그룹 이름 |
| 관리자 가상 PC 그룹 체크 (`grp_yn`) | 체크박스 | disabled |
| 네트워크 관리 (`networks`) | 테이블 + 설정 버튼 | 네트워크 선택/삭제. `networkSetting` 모달 |
| 기본 템플릿 정보 (`temp_nm`) | 읽기 전용 | 선택된 템플릿의 CPU/메모리/디스크/OS/IMAGE/SW 표시 |
| 가상 PC 정책 (`secu_plcy_nm`) | 읽기 전용 + 변경 버튼 | `virtualPcPolicySetting` 모달로 선택 |
| 백업/스냅샷 정책 (`bkup_snap_plcy_nm`) | 읽기 전용 + 변경 버튼 | `backupSnapshotPolicy` 모달로 선택 |

### 기본 템플릿 정보 (읽기 전용 표시)

| 항목 | 필드 |
|------|------|
| CPU | `templateData.vcpu_cnt` |
| 메모리 | `templateData.vmm_capa` |
| 디스크 | `templateData.vhd_capa` |
| OS | `templateData.os_typ_cd_nm` |
| IMAGE | `templateData.img_nm` + `img_id` |
| S/W | `templateData.sw_count` |

## 풀 상세 (VirtualPcPoolDetail)

풀 레벨에서 설정하는 항목. 풀 = 실제 VM이 생성되는 단위.

### 상태 정보 (상단 표시)

| 항목 | 필드 | 설명 |
|------|------|------|
| Pool 최대 수 | `max_vm_cnt` | 이 풀에 생성 가능한 최대 VM 수 |
| 사용자 할당 수 | `allo_vm_cnt` | 사용자에게 할당된 VM 수 |
| 생성된 VM 수 | `tot_vm_cnt` | 실제 생성된 VM 수 |
| 예약 수 | `cre_resv_vm_cnt` | 예약 대기 중인 VM 수 |
| 할당 준비 수 | `assign_ready_vm_cnt` | 할당 준비 완료된 VM 수 |
| 공용 사용자 VM 수 | `pool_ready_vm_cnt` | 공용 PC 풀 대기 중인 VM 수 |

↓ **Pool 상태 표시 (`vm_pool_sts_cd`)** — `isDev` 조건일 때만 노출. 상태 자동 동기화 (N초 간격).

### 설정 항목

| 항목 | 필드 | 설명 |
|------|------|------|
| 가상 PC 그룹명 | `vmGrpNm` | 읽기 전용 (상위 그룹명) |
| Pool명 (`vm_pool_nm`) | 입력 | 풀 이름 |
| 자동 할당 (`auto_allo_yn`) | 읽기 전용 | Y/N |
| AD 사용 (`ad_itlk_usg_yn`) | 읽기 전용 | Y/N |
| ↓ **ad_itlk_usg_yn = 'Y'일 때만 노출** | | |
| 컴퓨터 OU (`ad_vdi_ou`) | 읽기 전용 | AD 조직 단위 |
| Zone 선택 (`zone_nm`) | 드롭다운 | VM이 배치될 Zone |
| 상시 전원 ON (`alw_power_on_yn`) | 드롭다운 | Y: VM 항상 켜짐 / N: 사용 시만 |
| 가상 PC 유형 설정 | 라디오 | 전용/공용 PC에 따라 선택지 변경 (`computedVirtualType`) |
| ↓ **전용 PC (`policy_yn = 'Y'`)일 때** | | |
| 전용 PC 전원 정책 | 설정 버튼 | `privatePcPowerPolicy` 모달. 실행 이력(`commonPcPowerHistory`) 조회 가능 |
| ↓ **공용 PC (`policy_yn ≠ 'Y'`)일 때** | | |
| 초기화 정책 (`init_use_yn`) | 체크박스 | 공용 PC 반환 시 VM 초기화 여부 (`V001POO`일 때 disabled) |

### 풀과 전용/공용 PC의 관계

하나의 풀은 전용 **또는** 공용 중 하나. `tnt_mtd_cd` 값으로 결정:

| 풀 유형 | `tnt_mtd_cd` | 특징 |
|---------|-------------|------|
| 전용 풀 | `V001DED` | 사용자 고정 할당. 전원 정책 설정 가능. IP 고정/동적 선택 |
| 공용 풀 | `V001POO` | 풀에서 자동 배정. 초기화 정책 설정 가능. IP 동적 고정 |
