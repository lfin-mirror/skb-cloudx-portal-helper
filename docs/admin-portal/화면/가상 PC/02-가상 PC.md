# 가상 PC (T0702)

경로: /virtual-pc/virtual-pc-group

컴포넌트: views/virtualPc/VirtualPcGroup.vue

TA 전용 (SA 메뉴에 없음). `isSuperAdmin` 분기 없음.

그룹/풀에 할당된 **개별 VM 목록** 조회 및 관리. 전용/공용 PC 필터링, 실시간 동기화 지원.

## UI 레이아웃

상단 Pool 선택 영역 + 하단 VM 목록 테이블.

### 전용/공용 PC 필터

체크박스로 전용/공용 구분 조회:

| 체크박스 | `tnt_mtd_cd` 값 | 아이콘 |
|---------|----------------|--------|
| 전용 PC | `V001DED` | private 아이콘 |
| 공용 PC | `V001POO` | public 아이콘 |

기본값: 둘 다 선택 (`selectedOptions: ['V001DED', 'V001POO']`).

### Pool 선택 영역

| 항목 | 설명 |
|------|------|
| Pool 드롭다운 | 풀 목록 표시 (아이콘으로 전용/공용 구분) |
| PC 수량 표시 | 전체 PC / 전용 PC / 공용 PC 수량 |
| Pool 검색 | Pool명으로 검색 |

### 자동 갱신

두 가지 자동 갱신이 별도로 존재:

| 항목 | 대상 | 설명 |
|------|------|------|
| 재생/일시정지 버튼 + 반복 주기 선택 | **Pool 목록** | Pool 상태 자동 갱신 ON/OFF + 간격 선택 |
| 목록 동기화 토글 | **VM 목록** | VM 목록 실시간 동기화 ON/OFF |

### 테이블 컬럼

동적 컬럼 (`computedVPCCols`) — Pool 선택에 따라 컬럼 구성 변경. 주요 컬럼:

| 컬럼 | 필드 | 설명 |
|------|------|------|
| VM명 | `vm_nm` | — |
| Pool명 | `vm_pool_nm` | — |
| 사용자명 | `acct_nm` | — |
| 사용자 그룹 | `usr_grp_nm` | — |
| 가상 PC 상태 | `vm_power_sts_cd` | 상태 아이콘 |
| 접속 상태 | `usr_vm_conn_sts_cd` | 접속 아이콘 |

### 상태 범례

연결 품질 + PC 상태를 색상으로 구분하는 StatusPane.

### 액션 버튼 (테이블 컬럼)

개별 VM에 대한 일괄/단일 작업:

| 버튼 | 설명 |
|------|------|
| 정책 변경 | `VirtualPcPolicySettingForModal` 모달 → VM의 보안 정책 변경 |
| 전원 끄기 | VM 전원 OFF |
| 전원 켜기 | VM 전원 ON |
| 전원 리셋 | VM 재시작 |
| 사용기간 변경 | VM 사용 기간 연장/변경 |
| PC 정보 업데이트 | VM 정보 갱신 |
| 초기화 | VM 초기 상태로 복원 |

`VirtualCommonModal` — 전원 관련 일괄 처리 및 사용기간 변경 시 사용.

### 행 클릭 → 상세 (Split Panel)

행 클릭 시 `VirtualPcDetail` 컴포넌트가 Split Panel 하단에 표시. 개별 VM의 상세 정보 + 설정 변경.
