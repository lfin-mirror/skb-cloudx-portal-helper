# TA 메뉴 개요

Tenant Admin이 접근하는 admin-portal 메뉴 전체 현황.

## 대메뉴별 문서 현황

| 대메뉴 | 문서 위치 | 핵심 내용 |
|--------|----------|----------|
| 포털 (T04) | 화면/포털/ (9파일) | user-portal UI 커스터마이징, 공지/FAQ/1대1문의/매뉴얼/팝업/설치파일 |
| 테넌트 (T08) | 화면/테넌트/01-테넌트.md | 자기 테넌트 설정 수정 (삭제 불가) |
| 서비스 (T06) | 화면/서비스/02-서비스.md | 접근 통제, 외부 연동(AD/NAS/이메일/OCTATCO), Push 메시지 |
| 정책 (T05) | 화면/정책/ (8파일) | 테넌트별 정책 생성/관리, SA 템플릿 복사(정책 가져오기) |
| 템플릿 (T09) | 화면/템플릿/01-템플릿.md | SA와 동일 (isSuperAdmin 분기 없음, RBAC 제어) |
| 시스템 자원 (T10) | 화면/시스템 자원/ (5파일) | 자기 테넌트 범위 조회, 가상 네트워크 CRUD |
| 가상 PC (T07) | 화면/가상 PC/ (5파일) | 그룹/풀/VM 관리, 접속 조회, 예약, 포트/IP 관리 |
| 가상 디스크 (T04) | 화면/가상 디스크/01-가상 디스크.md | Cinder Volume 연결/분리/백업/복원 |
| 사용자 정보 (T02) | 화면/사용자 정보/ (3파일) | 사용자 그룹/계정 관리, 업무 요청, 단말 접속 |
| 사용자 지원 (T03) | 화면/사용자 지원/ (3파일) | 가상 PC 원격 제어, 업무 처리 요청, VOC |
| 관리자 (T12) | 화면/관리자/ (2파일) | 관리자 계정, 권한 그룹 |
