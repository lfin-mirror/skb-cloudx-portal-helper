# 정책 유저플로우

SA가 정책 템플릿을 정의하고, TA가 복사·수정하여 VM에 적용하는 전체 흐름.

---

## 정책 라이프사이클 개요

```
[SA] 정책 템플릿 정의 (*Supadm.vue)
  │  tgt_cd: U003S / U006S / U023S
  │  등록 버튼 없음 — 단일 정책 편집
  ▼
[TA] "정책 가져오기"로 SA 템플릿 복사
  │  tgt_cd: U003B / U006T / U023T
  │  복사된 정책을 테넌트 환경에 맞게 수정
  ▼
[TA] 가상 PC 그룹에 정책 연결
  │  그룹 설정에서 가상 PC 정책 / 백업 정책 선택
  ▼
[TA] 풀 → VM 생성
  │  그룹에 연결된 정책이 VM에 자동 적용
  ▼
VM 접속 시 정책 반영
```

| 단계 | 역할 | 화면 |
|------|------|------|
| 템플릿 정의 | SA | 각 정책의 `*Supadm.vue` |
| 정책 복사 | TA | 각 정책 목록 → "가져오기" 버튼 |
| 정책 수정 | TA | 각 정책 상세 폼 |
| 그룹 연결 | TA | [가상 PC 그룹](../admin-portal/화면/가상%20PC/01-가상%20PC%20그룹.md) (T0701) |
| VM 적용 | 시스템 | VM 생성/접속 시 자동 |

---

## 정책 종류별 흐름

### 1. 인증 정책

사용자/관리자 포털 로그인 시 적용되는 인증 수단 설정.

```
[SA] 인증 정책 템플릿 정의
  │  사용자 인증 (A0501): 로그인 방식, 2차 인증, 비밀번호 규칙
  │  관리자 인증 (A0502): 동일 구조, 대상만 admin
  ▼
[TA] 정책 가져오기 → 수정
  │  2차 인증 사용 여부, SMS/OTP 설정
  │  비밀번호 복잡도, 만료 기간
  │  VM 제한 수 (vm_rstt_cnt)
  ▼
[TA] 사용자 그룹에 정책 연결
  │  사용자 그룹 상세에서 인증 정책 선택
  ▼
사용자 로그인 시 정책 적용
  │  비밀번호 검증 → 2차 인증 → 접속 허용
```

| 단계 | 화면 | 비고 |
|------|------|------|
| SA 템플릿 | [인증 정책](../admin-portal/화면/정책/01-인증%20정책.md) (A0501, A0502) | 단일 폼 편집 |
| TA 정책 생성 | [인증 정책](../admin-portal/화면/정책/01-인증%20정책.md) (T0501, T0502) | 목록 + 가져오기 |
| 그룹 연결 | [사용자 그룹](../admin-portal/화면/사용자%20정보/01-사용자%20그룹.md) (T0201) | 인증 정책 드롭다운 |

### 2. 가상 PC 정책

VM 접속 시 적용되는 보안/동작 정책. 10개 설정 섹션의 허브.

```
[SA] 가상 PC 정책 템플릿 정의
  │  보안 항목 8개 (클립보드, USB, URL, 프린터 등)
  │  워터마크, 세션 만료, 자동 해상도 등
  ▼
[TA] 상세 정책 개별 생성 (선행 작업)
  │  ├── 예외 네트워크 (T050601)
  │  ├── 접근 차단 (T050602)
  │  ├── URL Redirection 금지 (T050606)
  │  ├── 미라이선스 정책 (T050607)
  │  ├── Metadata (T050608)
  │  ├── USB 정책 (T05060901)
  │  └── 접근 통제/보안 그룹 (T050611)
  ▼
[TA] 가상 PC 정책 생성 (가져오기 or 신규)
  │  10개 설정 섹션에서 위 상세 정책들을 드롭다운으로 선택
  │  ├── [1] 보안 정책 ← 예외 네트워크, USB, 보안 그룹
  │  ├── [6] 접근 차단 ← 접근 차단 정책 선택
  │  ├── [7] URL 예외 ← URL Redirection 정책 선택
  │  ├── [8] 미라이선스 ← 미라이선스 정책 선택
  │  └── [9] Metadata ← Metadata 정책 선택
  ▼
[TA] 가상 PC 그룹에 연결
  │  그룹 상세 → 가상 PC 정책 변경 모달
  ▼
VM 접속 시 정책 반영
```

| 단계 | 화면 | 비고 |
|------|------|------|
| SA 템플릿 | [가상 PC 정책](../admin-portal/화면/정책/02-가상%20PC%20정책.md) (A0503) | 단일 폼 |
| 상세 정책 생성 | [네트워크 보안](../admin-portal/화면/정책/04-네트워크%20보안%20정책.md), [SW 제어](../admin-portal/화면/정책/05-SW%20제어%20정책.md), [USB](../admin-portal/화면/정책/06-USB%20정책.md), [Metadata](../admin-portal/화면/정책/07-Metadata.md) | TA 전용 메뉴 다수 |
| 가상 PC 정책 | [가상 PC 정책](../admin-portal/화면/정책/02-가상%20PC%20정책.md) (T0503) | 목록 + 상세 폼 |
| 그룹 연결 | [가상 PC 그룹](../admin-portal/화면/가상%20PC/01-가상%20PC%20그룹.md) (T0701) | 정책 선택 모달 |

### 3. 백업/스냅샷 정책

VM 디스크의 자동 백업과 스냅샷 스케줄.

```
[SA] 백업/스냅샷 정책 템플릿 정의
  │  백업 주기, 보관 기간, 스냅샷 최대 수
  ▼
[TA] 정책 가져오기 → 수정
  │  테넌트 환경에 맞게 주기/보관 기간 조정
  ▼
[TA] 가상 PC 그룹에 연결
  │  그룹 상세 → 백업/스냅샷 정책 변경 모달
  ▼
VM 디스크 자동 백업/스냅샷 실행
```

| 단계 | 화면 |
|------|------|
| SA 템플릿 | [백업/스냅샷 정책](../admin-portal/화면/정책/03-백업%20스냅샷%20정책.md) (A0505) |
| TA 정책 생성 | [백업/스냅샷 정책](../admin-portal/화면/정책/03-백업%20스냅샷%20정책.md) (T0505) |
| 그룹 연결 | [가상 PC 그룹](../admin-portal/화면/가상%20PC/01-가상%20PC%20그룹.md) (T0701) |

### 4. 네트워크 보안 정책 (TA 전용)

SA 메뉴에 없는 TA 전용 상세 정책들. 가상 PC 정책에서 참조.

```
[TA] 상세 정책 생성
  │  예외 네트워크 (T050601) → IP 대역 그룹 정의
  │  접근 차단 (T050602) → 서비스별 IP/Port 차단 규칙
  │  접근 통제/보안 그룹 (T050611) → OpenStack Security Group 매핑
  ▼
[TA] 가상 PC 정책에서 참조
  │  [1] 보안 정책 → 예외 네트워크 그룹 선택
  │  [1] 보안 정책 → 접근 통제(보안 그룹) 선택
  │  [6] 접근 차단 → 접근 차단 정책 선택
  ▼
VM 접속 시 네트워크 보안 규칙 적용
```

| 화면 | 역할 |
|------|------|
| [네트워크 보안 정책](../admin-portal/화면/정책/04-네트워크%20보안%20정책.md) | 예외 네트워크, 접근 차단, 보안 그룹 |

### 5. SW 제어 / USB 정책 (TA 전용 다수)

```
[TA] SW 제어 정책 생성
  │  블랙리스트 (T050604) → 차단 프로세스
  │  화이트리스트 (T050605) → 허용 프로세스
  │  URL Redirection 금지 (T050606) → 차단 도메인/IP
  │  미라이선스 정책 (T050607) → 비인가 프로세스 차단
  ▼
[TA] USB 정책 생성
  │  USB 정책 (T05060901) → 허용/차단 규칙
  │  USB 유형/제조사/모델 (T05060902~04) → USB 분류 데이터
  ▼
[TA] 가상 PC 정책에서 참조
  │  [1] 보안 정책 → USB 정책 선택
  │  [7] URL Redirection → URL 정책 선택
  │  [8] 미라이선스 → 미라이선스 정책 선택
```

| 화면 | 역할 |
|------|------|
| [SW 제어 정책](../admin-portal/화면/정책/05-SW%20제어%20정책.md) | 블랙/화이트리스트, URL, 미라이선스 |
| [USB 정책](../admin-portal/화면/정책/06-USB%20정책.md) | USB 허용/차단 규칙 + 분류 데이터 |

## 정책 연결 전체 맵

```
SA 템플릿 정의                    TA 상세 정책 생성                TA 허브 정책 생성
──────────────                    ──────────────────              ──────────────────

인증 정책 (A0501/02)  ─복사─→  인증 정책 (T0501/02)  ──→  사용자 그룹 (T0201)
                                                              │
가상 PC 정책 (A0503)  ─복사─→  가상 PC 정책 (T0503)  ──→  가상 PC 그룹 (T0701)
                                     ▲                        │
                                     │ 참조                    │
                               예외 네트워크 (T050601)          │
                               접근 차단 (T050602)              │
                               URL 금지 (T050606)              │
                               미라이선스 (T050607)             │
                               Metadata (T050608)              │
                               USB 정책 (T05060901)            │
                               보안 그룹 (T050611)             │
                                                              │
백업/스냅샷 (A0505)   ─복사─→  백업/스냅샷 (T0505)   ──→  가상 PC 그룹 (T0701)
                                                              │
                                                              ▼
                                                         풀 → VM 생성
                                                         정책 자동 적용
```

## SA vs TA 역할 요약

| 역할 | SA | TA |
|------|----|----|
| 정책 생성 방식 | 단일 템플릿 편집 (등록/삭제 없음) | 목록 관리 (등록/삭제/가져오기) |
| 컴포넌트 | `*Supadm.vue` | `*.vue` |
| tgt_cd 접미사 | `S` (Super) | `B` (Base) / `T` (Tenant) |
| 상세 정책 접근 | Metadata + USB 4종만 | 전체 접근 |
| 정책 적용 | 직접 적용 불가 (템플릿 역할) | 그룹/풀/VM에 연결 |
| 사용 현황 | 비노출 | 그룹/풀/VM 수 확인 가능 |
