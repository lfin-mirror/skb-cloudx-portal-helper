# Admin Portal — SA 메뉴 트리

Super Admin이 접근하는 admin-portal 전체 메뉴 구조.

## 메뉴 트리

```
대시보드 (A01)                                          — 문서 없음
│
모니터링 (A03)                                          — 문서 없음
│
포털 (A04)
└── 관리자 UI (A0401)                                   → 문서 있음
│
정책 (A05)
├── 인증 정책 (A0501, A0502)                            → 문서 있음
├── 가상 PC 정책 (A0503)                                → 문서 있음
├── 백업/스냅샷 정책 (A0505)                             → 문서 있음
└── 상세 정책 (A0506)
    ├── Metadata (A050608)                              → 문서 있음
    ├── USB 정책 (A05060901)                            → 문서 있음
    ├── USB 유형 (A05060902)                            → 문서 있음
    ├── USB 제조사 (A05060903)                          → 문서 있음
    └── USB 모델 (A05060904)                            → 문서 있음
│
서비스 (A06)
└── 보안 인증서 (A0603)                                  → 문서 있음
    ├── 인증서 이력 (A060301)
    └── 인증서 등록 (A060302)
│
테넌트 (A08)
└── 테넌트 (A0802)                                      → 문서 있음
│
템플릿 (A09)
├── Flavor (A0901)                                      → 문서 있음 (통합)
├── Image (A0902)                                       → 문서 있음 (통합)
└── 템플릿 (A0903)                                      → 문서 있음 (통합)
│
시스템 자원 (A10)
├── 도메인 (A1001)                                      → 문서 있음
├── Zone (A1002)                                        → 문서 있음 (도메인과 통합)
├── 호스트 (A1003)                                      → 문서 있음
├── 네트워크 (A1004)                                    → 문서 있음
│   ├── 물리 네트워크 (A100401)
│   ├── 가상 네트워크 (A100402)
│   ├── 라우터 (A100403)
│   └── Floating IP (A100404)
├── 스토리지 (A1005)                                    → 문서 있음
│   ├── Volume Type (A100501)
│   ├── 디스크 QoS (A100502)
│   └── QoS Spec (A100503)
├── 마이그레이션 (A1006)                                → 문서 있음 (호스트와 통합)
├── 네트워크 스토리지 (A1007)                            → 문서 있음 (스토리지와 통합)
├── Proxy 서버 (A1009)                                  → 문서 있음
│   ├── Forward Proxy (A100901)
│   └── Reverse Proxy (A100902)
├── 이메일 (A1010)                                      → 문서 있음 (Proxy와 통합)
└── HA (A1011)                                          → 문서 있음 (호스트와 통합)
    ├── 대피 이력 (A101101)
    └── 대피 설정 (A101102)
│
통계 (A11)                                              — 문서 없음
│
관리자 (A12)
├── 관리자 계정 (A1201)                                  → 문서 있음
├── 시스템 설정 (A1204, A1205, A1206, A1209)             → 문서 있음
└── 메뉴/API/기능 관리 (A1208)                           → 문서 있음
    ├── 메뉴 관리 (A120801)
    ├── API 관리 (A120802)
    └── 기능 관리 (A120803)
```
