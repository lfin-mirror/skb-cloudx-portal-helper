# i18n (다국어) 구조

Vue-i18n 기반. 한국어(ko)가 기본 언어이고, 영어(en) 번역 파일이 존재하지만 현재 비활성화 상태다.

## 설정 — `src/lang/index.js`

```javascript
const i18n = new VueI18n({
  locale: localStorage.getItem('language') || 'ko',
  messages,
  silentTranslationWarn: true  // 누락된 번역 키 경고 억제
})
```

- 언어 설정은 localStorage `language` 키에 저장되어 세션 간 유지된다.
- Element UI 자체 로컬라이제이션은 주석 처리되어 있다 (커스텀 번역 사용).

## 번역 파일 구조

`src/lang/menus/` 아래 `ko/`와 `en/` 폴더에 동일한 구조로 파일이 존재한다:

| 파일 | 용도 | 주요 키 카테고리 |
|------|------|----------------|
| `page.js` | 페이지 제목/레이블 | common.label.user, common.label.pc, common.label.network, common.label.date, common.label.board |
| `message.js` | 안내/경고 메시지 | title, common, vnc, upload, virtualPc, login.error |
| `list.js` | 테이블 컬럼 레이블 | common.user, common.pc, common.network, common.date, monitoring |
| `button.js` | 버튼 텍스트 | common (search/register/create/save/cancel 등), login, virtualPc, policy |
| `form.js` | 폼 레이블 | prefix (sec/hour/year/month), label, label.statusLabel, label.ad |
| `validate.js` | 유효성 검사 메시지 | prefix (address/port/name/id/password), network, email, policy, virtualPc |
| `modal.js` | 모달 다이얼로그 | ip, virtualPcLicenseSetting, userSearch, certificationPolicySetting, sms 등 20+ |
| `disk.js` | 디스크 관리 | form.label, form.select.option, modal.title, modal.alert, alert |

## 키 구조 패턴

중첩된 객체 구조를 사용한다. 최대 4~5단계 깊이:

```
파일 카테고리 → 기능 그룹 → 세부 항목
button.common.search      → '조회'
message.login.error.user_not_exist → '존재하지 않는 사용자입니다.'
list.monitoring.listContainer.searchFilter.searchWord → '검색어'
```

파라미터 치환을 지원한다:
```javascript
message.upload.allowFileType = '{0} 확장자만 등록 가능 합니다.'
```

## 사용 방식

```vue
<!-- 템플릿 -->
{{ $t('button.login.login') }}

<!-- 스크립트 -->
this.$t('message.common.saveComplate')
```

## 언어 전환

Vuex `app` 모듈의 `setLanguage` 액션으로 처리한다:
1. localStorage에 언어 코드 저장
2. `<html lang>` 속성 업데이트
3. 세션 관련 코드 초기화

현재 배포 환경에서는 한국어만 활성화되어 있다. 영어 번역 파일은 `index.js`에서 import가 주석 처리되어 있어 사용되지 않는다.
