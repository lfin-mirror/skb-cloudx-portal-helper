# 상태 코드 및 에러 코드

프로젝트 전반에서 사용되는 코드값 정의.

## 사용자 상태 코드 (acct_sts_cd)

| 코드 | 의미 |
|------|------|
| `U013NR` | 정상 |
| `U013SL` | 휴면 |
| `U013IN` | 초기화 (initialized) |

## 2차 인증 유형 코드 (n2nd_cert_typ_cd)

| 코드 | 인증 수단 |
|------|----------|
| `U004SMS` | SMS |
| `U004EML` | 이메일 |
| `U004OTP` | Google OTP |
| `U004KKT` | 카카오톡 |

## 인증 에러 코드 (AUTH-xxxx)

| 코드 | 의미 |
|------|------|
| `AUTH-4211` | 기존 비밀번호 불일치 |
| `AUTH-4214` | 인증번호 불일치 |
| `AUTH-4216` | 인증 세션 만료 |
| `AUTH-4218` | Octatco 인증 실패 |
| `AUTH-4219` | 유효하지 않은 전화번호 |
| `AUTH-4220` | 과도한 인증번호 발송 (30분 제한) |
