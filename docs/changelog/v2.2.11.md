# v2.2.11 — 보안그룹(접근 통제) 신규 + 2차인증 이메일 추가

날짜: 2026-04-13

## 개요

접근 차단을 대체하는 보안그룹(접근 통제) 기능 신규 추가. 관리자 2차인증 이메일 수단 추가, 가상PC 에이전트 로그 다운로드 기능 추가. 키로깅 방지 관련 버그픽스 다수 포함.

## admin-portal 화면 문서 (수정 4개, 신규 1개)

| 문서 | 변경 내용 |
|------|----------|
| 정책/04-네트워크 보안 정책 | 접근 차단 deprecated + 보안그룹 목록/등록/상세 4탭/Rule 추가 모달 신규 통합 |
| 정책/01-인증 정책 | 2차인증 인증서 설정 섹션 추가 (SMS/EMAIL) |
| 가상 PC/02-가상 PC | 에이전트 로그 다운로드 버튼 + 확인 모달 추가 |
| 테넌트/01-테넌트 | klp_os_allow_yn 서버 미전송 사실 보충 |
| term/02-keylogging (신규) | admin-portal 키로깅 방지 2단계 제어 구조 (시스템 설정 전역 + 테넌트별) |
